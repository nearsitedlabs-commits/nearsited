## **Nearsited Brand North Star** 

Nearsited is Opportunity Intelligence for Web Agencies. 

Core belief: 

The best opportunities are not hidden. They are overlooked.

Design Goals 

1. Premium 

Feels expensive. 

1. Product First 

Show software before marketing. 

1. High Information Density 

Optimized for agency workflows. 

1. Calm 

Avoid visual noise. 

1. Fast 

Interfaces should feel responsive.

Landing Page 

Purpose: 

Create confidence. 

Show the product. 

Demonstrate outcomes.

"The front door of the application." 

Application 

Purpose: 

Help agencies discover and close opportunities. 

Optimize for speed and clarity. 

Shared Design Language 

Landing page and SaaS must share: 

• colors • typography • spacing • radii • shadows • buttons • cards • motion 

A user should experience zero visual surprise after signing in.
